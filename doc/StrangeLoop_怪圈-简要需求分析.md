# Strange Loop-怪圈 简要需求分析



## 一、项目概述



### 项目名称：  

​	Strange Loop 怪圈



### 项目背景：

​	随着人工智能技术的迅猛发展，尤其是大语言模型（LLM）和生成式AI的广泛应用，越来越多的企业和开发者开始部署和使用这些强大的模型。大模型不仅能够提供高效的自然语言处理、机器翻译、文本生成等服务，还能够在图像生成、语音识别、智能决策等领域展现出强大的应用潜力。然而，这些模型通常体积庞大、计算密集，且需要精细化的管理和调度，以实现其在不同场景下的最优表现。



### 项目目标：

​	怪圈项目旨在建立一个高效、灵活的大模型管理与应用平台，通过统一的管理、调度和优化手段，帮助企业和开发者更好地利用大模型技术，降低大模型的使用门槛，推动大模型的实际应用落地，组合出各种各样的有趣实用的功能。

​	同时，通过这个项目，也能使我们学到更前沿的技术，帮助我们未来的就业，和职业选择。



### 技术背景：

#### **后端架构**

- **框架与技术栈**：
  - **Spring Boot**：作为后端开发的核心框架，简化了应用的配置和部署。提供高效的开发、自动配置功能，使得构建可扩展的微服务架构更加容易。
  - **MySQL**：作为关系型数据库，用于存储项目中的核心数据（如用户信息、模型配置、任务记录等）。MySQL 具备高效的查询能力，适用于平台中大部分结构化数据的存储需求。
  - **MyBatis**：作为数据库操作框架，与 MySQL 集成，通过映射文件提供 SQL 语句与 Java 方法的映射，简化了数据库操作，提升了开发效率。
  - **Spring Security**：用于处理身份验证与授权，保障平台的数据安全。通过基于角色的访问控制 (RBAC) 和用户认证机制，确保敏感数据的安全性。
- **业务逻辑与模块化设计**：
  - **用户与权限管理**：利用 Spring Security 实现基于角色的访问控制 (RBAC)，确保不同权限的用户对系统的访问有明确限制。
  - **模型管理与调度**：后端管理大模型的注册、配置、版本管理、资源调度等功能，确保模型能够高效、灵活地服务于不同的任务。
  - **任务调度与队列**：通过集成消息队列（如 RabbitMQ）和调度框架（如 Quartz），实现任务的异步执行与任务队列管理。
  - **日志与监控**：结合 Spring AOP 实现请求日志、错误日志记录，并利用 ELK（Elasticsearch、Logstash、Kibana）进行系统监控，实时分析后端性能。

#### **前端架构**

- 框架与技术栈：
  - **Vue3**：作为前端开发的核心框架，利用 Vue3 提供的 Composition API，增强了组件化开发的灵活性。Vue3 支持高效的虚拟 DOM 渲染，使得用户界面更加流畅。
  - **Element UI**：用于构建前端界面，提供丰富的 UI 组件库，简化了开发周期，并且具有高度的可定制性，能满足怪圈项目对后台管理界面的需求。
- 前端功能模块：
  - **管理后台**：前端通过 Vue3 和 Element UI 构建管理界面，提供模型管理、任务调度、资源监控、权限管理等功能，支持高效的操作与实时反馈。
  - **任务监控**：提供实时数据流监控和统计信息，前端展示后端返回的任务状态、资源使用情况、模型执行进度等信息，确保平台稳定运行。
  - **API 集成**：前端通过 Axios 等 HTTP 客户端调用后端 RESTful API，与后端进行数据交互。API 包括用户认证、模型管理、任务提交等。
  - **响应式设计**：前端支持响应式布局，在不同屏幕尺寸下能自适应展示，保证管理界面在不同设备上的良好体验。



### 主要功能模块简述：

- **用户管理模块**

  - **用户注册：**支持用户的注册
  - **用户登录：**支持用户的登录功能
  - **个人信息管理：**支持用户对个人信息的管理
  - **用户权限管理：**支持用户权限的管理

- **模型管理模块**

  - **大模型的增删改查：**支持对大模型的创建、修改、删除、查询
  - **对特殊模型的创建：**支持用户创建一些特殊模型（ps：如专注与扮演角色的模型、专注于对一个模拟环境输出的模型、专注于对一个故事进行主持的模型）
  - **支持用户上传各种模型：**用户可以上传使用本地Ollama管理的模型，也可以直接上传模型的API来进行管理。

- **模型应用模块：**

  - **模型聊天功能：**可以直接于构建好的模型进行对话。

  - **文字游戏功能：**可以让用户选择构建好的模型参与文字游戏的扮演。可以选择模型对文字游戏中的环境、角色、主持进行单独的扮演。

  - **文生图功能：**可以让构建的文生图模型在这里进行文生图操作。

  - **AI组件低代码生成模块：**可以在这里进行一些AI组件自定义，和一键生成导出Vue2或者Vue3版本代码，可以方便用户在编写自己的项目的时候更方便的使用AI技术。
